### 6. If a column has continuous data with many distinct values then we should prefer bucketing over Partitioning.

**Answer:**
In the above scenario, the cost of partitioning the data will be high because the data has many unique values which will lead to creation of too many partitions. This will create massive overhead. Instead, bucketing can be done using a hash function over data values.

### 7. Assuming HDFS block size to be 128MB, what will be the number of buckets for a file of size 2GB?

**Answer:**
Number of buckets = **2048MB / 128MB = 16**
